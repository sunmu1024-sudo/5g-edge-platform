# 🌐 5G边缘计算平台 - 智能传感器监控系统

> 蚌埠学院物联网工程专业 - 真实传感器数据监控平台

## 🚀 项目简介

这是一个基于Python Flask的5G边缘计算平台，专门用于连接和管理真实物理传感器。平台提供实时数据监控、设备管理、AI智能分析等功能。

## ✨ 核心特性

- 🔌 **多传感器支持** - 温度、湿度、光照、压力、摄像头等
- 🧠 **AI智能分析** - 数据预测、异常检测、优化建议
- 📊 **实时监控** - 实时数据流和图表展示
- 🖥️ **边缘计算** - 分布式节点管理
- 🎨 **现代化UI** - 响应式设计，支持明暗主题
- 🔒 **数据安全** - 混合存储方案（SQLite + JSON）

## 🛠️ 技术栈

**后端**: Python + Flask + SQLite  
**前端**: HTML5 + CSS3 + JavaScript + ECharts  
**AI引擎**: 机器学习预测算法  
**部署**: 本地服务器 + ngrok内网穿透
